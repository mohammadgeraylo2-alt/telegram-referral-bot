# 🎬 CinemaVault | ربات فیلم و سریال

ربات تلگرامی برای اشتراک‌گذاری فیلم/سریال با لینک اختصاصی، جوین اجباری، و پنل ادمین.

## امکانات
- جوین اجباری به دو (یا بیشتر) کانال، قابل مدیریت با دستور
- فوروارد فایل توسط ادمین → ساخت خودکار لینک دیپ‌لینک
- حالت بچ: چند فایل با هم زیر یک لینک (مثلاً چند قسمت یه سریال)
- مدیریت ادمین‌ها و کانال‌های جوین اجباری از داخل خود ربات، بدون نیاز به تغییر کد
- آمار کاربران و ارسال پیام همگانی (Broadcast)
- ذخیره‌سازی فایل‌ها در یک کانال خصوصی (نه دیتابیس، پس حجم فایل مهم نیست)

## پیش‌نیازها
1. یک **BOT_TOKEN** از @BotFather (دستور `/newbot` رو بزن و اسم/یوزرنیم انتخاب کن)
2. یک **کانال خصوصی** جدید فقط برای ذخیره‌سازی فایل‌ها (برای خودت، نه برای کاربرا) و ربات رو توش **ادمین** کن.

> نیازی به my.telegram.org و API_ID/API_HASH نیست — این نسخه فقط با توکن ربات کار می‌کنه.

## نصب محلی (تست)
```bash
git clone <آدرس-ریپازیتوری-شما>
cd movie-bot
pip install -r requirements.txt
cp .env.example .env
# فایل .env رو با اطلاعات خودت پر کن
python bot.py
```

## دیپلوی روی Railway
1. این پروژه رو توی یه ریپازیتوری گیت‌هاب آپلود کن (مراحل پایین).
2. وارد https://railway.app بشو → New Project → Deploy from GitHub repo → ریپازیتوری رو انتخاب کن.
3. توی تب **Variables** پروژه‌ی Railway، همون متغیرهای داخل `.env.example` رو با مقادیر واقعی اضافه کن:
   - `BOT_TOKEN`
   - `STORAGE_CHANNEL` (آیدی عددی کانال خصوصی؛ با فوروارد یه پیام از اون کانال به @userinfobot می‌تونی آیدیش رو پیدا کنی)
   - `BOT_USERNAME` (یوزرنیم ربات بدون @)
4. Railway به‌صورت خودکار `requirements.txt` و `Procfile` رو تشخیص میده و ربات رو اجرا می‌کنه.
5. اگه دیتابیس بین دیپلوی‌ها ریست شد، از بخش **Volumes** توی Railway یه Volume به مسیر پروژه وصل کن تا فایل `bot.db` پایدار بمونه.

## آپلود روی گیت‌هاب
```bash
cd movie-bot
git init
git add .
git commit -m "اولین نسخه ربات"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```
⚠️ فایل `.env` هیچ‌وقت آپلود نمیشه (توی `.gitignore` هست) — توکن و کلیدهات رو مستقیم توی گیت‌هاب نذار.

## استفاده روزمره (پنل ادمین)
- یه فیلم رو مستقیم برای ربات فوروارد کن → لینک تکی فوری میگیری.
- برای چند فایل زیر یه لینک: `/newbatch` → فایل‌ها رو یکی‌یکی فوروارد کن → `/donebatch`
- `/channels` `/addchannel` `/delchannel` برای مدیریت جوین اجباری
- `/admins` `/addadmin` `/deladmin` برای مدیریت ادمین‌ها
- `/stats` آمار کلی
- `/broadcast` (روی یه پیام ریپلای کن) پیام همگانی
- `/help` لیست کامل دستورات

## نکته درباره‌ی محتوا
مسئولیت محتوایی که از طریق این ربات به اشتراک گذاشته میشه (رعایت قوانین کپی‌رایت و قوانین تلگرام) بر عهده‌ی مدیر رباته.
